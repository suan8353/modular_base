# 贡献指南

感谢你对 ModularBase 的关注！项目目前处于早期开发阶段，非常欢迎社区参与。

## 当前阶段可以做什么

### 🌟 关注和反馈

- **Star** 项目，关注进展
- 阅读 [架构设计文档](architecture.md)
- 在 Issue 中讨论架构设计、提出建议

### 💡 讨论和建议

我们特别欢迎以下方面的讨论：

- 架构设计的改进建议
- 数据包规范的完善
- 训练策略的优化
- 潜在的应用场景

### 📝 文档完善

- 修正文档中的错误
- 补充说明和示例
- 翻译文档（英文版）

## 后续阶段

随着项目推进，将开放更多贡献方式：

- **代码贡献**：核心模块优化、新功能实现
- **数据包开发**：贡献领域数据包
- **测试和验证**：不同硬件环境的测试

## 提交 Issue

提交 Issue 时请包含：

1. **问题类型**：Bug / 功能建议 / 讨论
2. **详细描述**：清晰描述问题或建议
3. **相关背景**：如果是技术讨论，请说明你的思考

## 代码规范

- Python 代码遵循 PEP 8
- 函数和类需要 docstring
- 关键逻辑需要注释
