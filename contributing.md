## Submitting Issues

When submitting an Issue, please include:

1. **Issue Type**: Bug / Feature Request / Discussion
2. **Detailed Description**: Clearly describe the issue or suggestion
3. **Background**: For technical discussions, explain your thinking

## Code Standards

- Python code follows PEP 8
- Functions and classes need docstrings
- Key logic needs comments
